## Required Image Specifications

### Hero Section
- **File**: `assets/images/hero-poster.jpg`
- **Dimensions**: 1920 x 1080 pixels (minimum)
- **Purpose**: Fallback image for hero video
- **Style**: High-quality photo of printing machines or facility
- **Format**: JPEG, high quality

### Main Page Images
- **Dimensions**: 800 x 600 pixels (4:3 aspect ratio)
- **Format**: JPEG, high quality (85% quality setting)
- **Style**: Professional, well-lit photographs

### Gallery Images
- **Dimensions**: 800 x 600 pixels (4:3 aspect ratio)
- **Format**: JPEG, high quality
- **Recommended Quantity**: 12 images minimum
- **Categories to cover**:
  - Banners and large format displays
  - Printing machines (CTCP, Large Format, DI)
  - Embroidery samples on apparel
  - DTF printing on fabrics
  - ID cards and identification materials
  - Business cards and stationery
  - Event banners and promotional materials

## Video Requirements

### Hero Background Video
- **File**: `assets/video/hero-video.mp4`
- **Dimensions**: 1920 x 1080 pixels (Full HD)
- **Duration**: 15-30 seconds, seamless loop
- **File Size**: Maximum 10MB (for fast loading)
- **Format**: MP4, H.264 codec
- **Audio**: Muted (no sound required)
- **Content**: Professional printing machines in action (CTCP, large format, digital printing, embroidery)
- **Style**: Clean, professional footage with good lighting

### Video Optimization Tips
1. Use Handbrake or similar software to compress
2. Remove audio track
3. Target bitrate: 2-3 Mbps for web
4. Ensure seamless loop (first and last frames should match)

## Professional Photography Tips

### For Printing Machines
1. Clean machines before shooting
2. Use natural or professional studio lighting
3. Capture machines from multiple angles
4. Show machines in operation if possible
5. Include close-ups of printing details

### For Finished Products
1. Use neutral background
2. Ensure even lighting
3. Photograph on clean surfaces
4. Include scale reference for large items
5. Capture detail shots and wide shots

## Image File Naming Convention
Use descriptive names:
- `banner-exhibition-01.jpg`
- `machine-ctcp-01.jpg`
- `embroidery-uniform-corporate.jpg`
- `id-card-staff-sample.jpg`

Avoid generic names like `image001.jpg` or `IMG_1234.jpg`
